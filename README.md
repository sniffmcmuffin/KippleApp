# KippleApp

ThemeMode
- Byta tema genom att använda ThemeService.SetTheme("theme-dark") och JS.InvokeVoidAsync("setTheme", theme), tex en knapp.
- Använd inte hårdkodade färger.
- Använd alltid CSS-variabler, tex background: var(--color-bg);
- Lägga till nya variabler genom att lägga till dem i båda temablocken i app.css.
